# Менеджер паролей

## Стек

_Frontend_

- React
- JavaScript
- TailwindCSS
- React Toastify

_Инструкция по запуску приложения_

- Склонировать репозиторий
- Выполнить в терминале npm i для установки зависимостей
- Запустить клиентское приложение с помощью команды npm run dev в терминале
